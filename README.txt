1. Cambiar en el archivo server.js usuario y contraseña de mysql.
2. Ejecutar el comando npm install para instalar los paquetes que se utilizaron para esta prueba.
3. Ejecutar npm start en el directorio raiz para la ejecucion del backend.
4. Las vistas de login y registro no se ejecutan en servidor entonces debera
   abrirlas de forma local usando la ruta absoluta en el navegador.
5. (OPCIONAL) Registrar un usuario antes de probar el login.
6. Existen validaciones de campos vacios, usuario unico y la contraseña es de minimo 8 caracteres.
7. La contraseña tiene un metodo de encriptacion sha256 irreversible.
